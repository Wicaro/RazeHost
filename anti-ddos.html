Como funciona o nosso Anti-DDoS?

Com 2 Tbps de capacidade, os scrubbing centers da RazeHost estão equipados para lidar com qualquer ataque distribuído, independentemente do seu tipo ou forma. Provedores que não têm capacidade suficiente, muitas vezes, mudam o apontamento da rota quando você está sofrendo um ataque. Em vez de rotear, a RazeHost irá mitigar, o que significa nenhum aumento na latência para você..

Out of Path

A Raze Host padroniza uma topologia Out-of-Path para que a infraestrutura de rede seja escalonável sem depender do dispositivo de mitigação e, mais importante, para que o dispositivo de mitigação não se torne um ponto de falha único. Quando um IP está no modo Out-of-Path, o appliance só começa a funcionar quando necessário.

InLine

Disponível no pacote Anti-DDoS Pro, os IPs podem ser definidos como Inline para serem protegidos todos os momentos, independentemente de um ataque estar realmente acontecendo. No modo Inline, todo o tráfego desse IP passa pelo centro de depuração Raze Host mais próximo disponível. A proteção começa instantaneamente quando um ataque é identificado.
